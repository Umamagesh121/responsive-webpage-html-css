# 🌐 Responsive Website with Contact Form

A simple and responsive website created using **HTML5** and **Internal CSS**. This project contains a responsive homepage with a navigation bar, sidebar, animated images, and a comprehensive contact form demonstrating various HTML form elements.

---

# 📌 Features

* ✅ Responsive Web Design
* ✅ HTML5 Semantic Structure
* ✅ Internal CSS Styling
* ✅ Sticky Navigation Bar
* ✅ Responsive Flexbox Layout
* ✅ Sidebar and Main Content
* ✅ Image Animation using CSS Keyframes
* ✅ Modern Contact Form
* ✅ Gradient Background
* ✅ Rounded Input Fields
* ✅ Hover Effects
* ✅ Input Focus Effects
* ✅ Mobile-Friendly Design
